# Action Type: Send_Response_Code

Action ID: **22**

## Purpose

Sends a response code (optionally symptom scoped).

## Value fields

* Value1: Response code
* Value2: Optional symptom code
* Value3: not used

* Configure this action in the hyperlink actions editor from **Guide Card Editor**.
