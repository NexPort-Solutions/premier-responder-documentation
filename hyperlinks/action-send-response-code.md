# Action Type: Send_Response_Code

Action ID: **22**

## When to use

Use when clicking a hyperlink should send a response-code driven dispatch update.

## Fields In Action Editor

- **Response Code:** Enter the response code to apply.
- **Symptom Code (optional):** Optionally enter a symptom code/text.

## What The Call-Taker Sees

Dispatch response processing runs using the entered response code (and optional symptom code).

## Example Setup

- Response Code: ALPHA
- Symptom Code (optional): CHEST
