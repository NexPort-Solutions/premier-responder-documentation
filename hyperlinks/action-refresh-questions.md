# Action Type: Refresh_Questions

Action ID: **4**

## When to use

Use when clicking a hyperlink should re-run General Question action logic.

## Fields In Action Editor

- This action does not show editable fields in the action editor.

## What The Call-Taker Sees

All configured General Question actions are processed again for the current call state.

## Example Setup

- No field entries are required for this action.
