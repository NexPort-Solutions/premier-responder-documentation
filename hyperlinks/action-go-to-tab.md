# Action Type: Go_To_Tab

Action ID: **13**

## Purpose

Navigates to a specific tab in the active call UI.

## Value fields

* Value1: Tab identifier
* Value2: not used
* Value3: not used

## Editor notes

* Configure this action in the hyperlink actions editor from **Guide Card Editor**.
