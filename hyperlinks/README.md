# Hyperlinks

This section documents hyperlink action configuration in **Premier Responder Administrator**.

## What this section includes

* An action-type reference for every hyperlink action type supported by the action model.
* Field mapping guidance for `Value1`, `Value2`, and `Value3`.
* Cross-links to editors where hyperlink actions are configured.

## Where hyperlink actions are configured

1. Open **Guide Card Editor**.
2. Select hyperlink text in a guide-card pane.
3. Open the actions editor and choose an **Action Type**.
